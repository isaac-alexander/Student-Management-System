OBJECTIVE
Build a simple Student Management System web application using Spring Boot, Spring MVC, Thymeleaf, Spring Data JPA (Hibernate), MySQL


Tools and technologies used
Latest Java
Spring Boot
Spring MVC
Spring Data JPA (Hibernate)
MySQL
Thymeleaf
Eclipse STS
Bootstrap

Start up spring project
1. Go to https://start.spring.io/

2. Name the group, artifact and package name

3. Select Java version

4. Add all necessary dependencies (Spring Boot Dev Tools, Spring Data JPA, MySQL Driver, Thymeleaf, Spring Web etc...)

5. Click on Generate

6. Extract file

7. Open folder and Open pom.xml

8. Create controller package

9. Create service package

10. Create repository package

11. Create entity package

12 . Configure MySQL database

Steps needed for sql

1. create database in mysql server/ mysql workbench (create database (database name)

2. Go to src/main/resources/application.properties and configure mysql details
spring.datasource.url=jdbc:mysql://localhost:3306/sms?useSSL=false&serverTimezone=UTC&useLegacyDatetimeCode=false
spring.datasource.username=root
spring.datasource.password=###

3. Configure #Hibernate
#Hibernate
spring.jpa.database-platform=org.hibernate.dialect.MySQLDialect

#Hibernate auto ddl
spring.jpa.hibernate.ddl-auto=update (creates table in the database if it does not exist and updates table

logging.level.org.hibernate.SQL=DEBUG
